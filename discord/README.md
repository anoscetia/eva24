# eva24

a dark evangelion-palette theme for [discord](https://discord.com).

part of the [eva24 collection](https://github.com/anoscetia/eva24).

## install

### betterdiscord

1. clone this repo (or download `discord/eva24.theme.css`)
2. copy `eva24.theme.css` into your betterdiscord themes folder:
   - **linux**: `~/.config/BetterDiscord/themes/`
   - **macos**: `~/Library/Application Support/BetterDiscord/themes/`
   - **windows**: `%appdata%\BetterDiscord\themes\`
3. open discord → `settings → themes` and toggle **eva24**

the theme imports `system24` from cdn, so no extra setup is needed.

## font

ships with [dm mono](https://fonts.google.com/specimen/DM+Mono), applied to interface and code blocks.
