## install

### from source

1. clone this repo
2. copy `obsidian/theme.css` into `<vault>/.obsidian/themes/eva24/theme.css`
3. in obsidian, open `settings → appearance → themes` and pick **eva24**

## font

ships with [dm mono](https://fonts.google.com/specimen/DM+Mono) loaded via google fonts, applied to text, interface, and monospace.

## palette variants

swap the accent palette by adding a class to the body (via a css snippet or theme toggle). defaults to rose-pine.

| variant | class |
|---|---|
| rose-pine (default) | — |
| rose-pine-moon | `rose-pine-moon` |
| nord | `nord-dark` |
| catppuccin mocha | `catppuccin-mocha` |
| everforest | `everforest-dark` |
| biscuit | `biscuit-dark` |
| obsidian | `obsidian-dark` |
| uw deep | `uw-deep` |
| uw ocean | `uw-ocean` |
| uw seaweed | `uw-seaweed` |
| uw sand | `uw-sand` |
