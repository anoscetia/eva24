# eva24

a dark evangelion-palette theme for [zed](https://zed.dev)

part of the [eva24 collection](https://github.com/anoscetia/eva24)

## install

### from source (dev extension)

1. clone this repo
2. in zed, open the command palette and run `zed: install dev extension`
3. select the `eva24/zed` directory
4. open theme selector and pick **eva24**
